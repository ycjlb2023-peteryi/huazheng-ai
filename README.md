# 铧正外贸AI技术方案系统

> 保定市铧正电气 - 专业技术方案数字员工

## 📋 项目简介

面向外贸业务员的 PC 端 AI 应用，基于企业私有技术资料与国家标准，自动理解客户定制需求，**秒级生成专业技术方案**，并输出客户可读的中英文回复内容。

### 核心价值

- ⚡ **秒级响应**：3分钟内生成专业技术方案
- 🎯 **精准方案**：基于企业知识库与国家标准
- 🌐 **双语输出**：自动生成中英文客户回复
- 📚 **知识沉淀**：企业核心能力数字化

## 🚀 技术栈

- **框架**：Angular 20+
- **语言**：TypeScript (strict mode)
- **状态管理**：Angular Signals
- **样式**：SCSS + CSS Variables
- **架构**：Standalone Components
- **控制流**：@if, @for (Angular 新语法)

## 🎨 核心功能

### 1. 登录认证
- 企业账号登录
- 角色权限管理（管理员/业务员）
- 精美渐变背景设计

### 2. 工作台
- 数据统计卡片
- 快速操作入口
- 最近方案展示
- 使用小贴士轮播

### 3. 方案生成（核心功能）
- **输入阶段**
  - 客户需求输入（支持中英文）
  - AI提示问题建议
  - 语言选择（中文/英文）
  
- **生成阶段**
  - 实时AI生成状态
  - 3分钟内完成
  
- **结果展示**
  - 左侧：技术方案（内部版）
    - 产品判断
    - 推荐配置（标配/选配）
    - 技术依据
    - 待客户确认问题
  - 右侧：客户回复内容（可直接复制）
    - 一键导出Word/PDF
    - 复制文本功能

### 4. 历史记录
- 方案列表展示
- 搜索和筛选
- 状态管理（已完成/草稿）
- 统计数据展示

### 5. 知识库管理（管理员专属）
- 文档上传（PDF/Word/Excel）
- 分类管理
- 优先级设置
- 适用产品配置
- 文档状态控制

## 🎓 核心概念

### AI 数字员工定位
本系统中的 AI 不是"工具"，而是**可被信任的数字员工**：
- 身份：企业内部最优秀的"技术方案官"
- 知识来源：企业私有资料 + 国家标准
- 工作对象：外贸业务员
- 工作目标：快速给出"可发给客户的方案"

### 产品边界（V1.0）
**支持：**
- ✅ 定制类高客单产品技术方案
- ✅ 中英文客户回复
- ✅ 企业知识库 + 国标依据

**不支持：**
- ❌ 自动报价
- ❌ 自动下单
- ❌ 全品类覆盖
- ❌ 开放联网搜索
